哈萨克语 AI 网站访问说明

本地访问地址：
http://localhost:8787

发布部署说明：
请查看同目录下的“部署发布说明.txt”

使用方法：
1. 第一次使用，请双击“启动网站.bat”。
2. 如果自动弹出 .env 文件，请把 DEEPSEEK_API_KEY= 后面改成你的 DeepSeek API Key，然后保存关闭。
3. 再次双击“启动网站.bat”，浏览器会自动打开网站。
4. 以后使用时，只需要先双击“启动网站.bat”，再打开上面的地址即可。

注意：
- 不要把 .env 文件发给别人，因为里面会保存你的 API Key。
- 如果提示没有 npm 或 node，请先安装 Node.js。
- 朗读功能建议使用最新版 Edge 或 Chrome 浏览器。
